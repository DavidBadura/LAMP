LAMP

Features
--------

* Apache2
* mySQL
* php5
* phpmyadmin
* samba
* nodejs
* git
* composer
* vim

---

Leider kann die vim Konfiguration nicht automatisch installiert werden.
Um die vim konfiguration zu installieren führe einfach diesen befehl `bash ./vim_install.sh` im home Verzeichnis aus.
